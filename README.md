# react-flask-nginx

## Install/Setup

Setup python env
```
python3 -m venv env
source env/bin/activate
```

Install dependencies
```bash
npm i
npm run setup
```

## Development

Run locally 
```
npm run dev
```

Navigate to `http://localhost:81/`

## Production

Run in production
```
npm run build
npm run prod
```

Navigate to `http://localhost:81/`
